# Plant Website – Client-Side Development

A responsive multi-page plant-themed website developed as part of the Client-Side Development module.  
This project demonstrates front-end development skills using HTML, CSS, and JavaScript, with a clean and well-organized structure.

---

## Project Overview

This website was designed and developed to demonstrate:

- Structured front-end development practices
- Clean file and asset organization
- Responsive layout design
- Multi-page website navigation
- Basic JavaScript interactivity

The website includes the following pages:

- Home
- About
- Collections
- Contact
- Testimonials

---

- HTML5
- CSS3
- JavaScript
- Git & GitHub
